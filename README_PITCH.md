# 💼 Job Application Tracker

A full stack web application that helps job seekers organize, track, and manage their job applications in one centralized place.

---

## 🚀 Overview

Job searching can quickly become overwhelming. Many people rely on spreadsheets, notes apps, emails, and memory to keep track of applications, interviews, and recruiter conversations.

**Job Application Tracker** solves this by providing a structured, intuitive platform to manage the entire job search pipeline—from application to offer.

---

## ❗ Problem

Job seekers often struggle to:
- Track where they’ve applied
- Remember application statuses
- Store recruiter/contact information
- Keep notes from interviews
- Understand overall job search progress

Traditional tools like spreadsheets are not built around the hiring workflow.

---

## 💡 Solution

This app provides a centralized system where users can:
- Create and manage job applications
- Track application status (Applied → Interview → Offer → etc.)
- Store notes and recruiter details
- Visualize progress through a dashboard
- Filter and search applications easily

---

## 👥 Target Users

- Active job seekers
- Bootcamp graduates
- College students
- Professionals exploring new opportunities

---

## ✨ Features

### 🔐 Authentication
- Secure user registration and login
- JWT-based authentication
- Protected routes

### 📋 Application Management
- Create, read, update, and delete job applications
- Store:
  - Company name
  - Role
  - Status
  - Application date
  - Job URL
  - Notes
  - Recruiter contact info

### 📊 Dashboard
- Visual overview of job applications
- Track progress across hiring stages

### 🔎 Filtering & Search
- Quickly find applications by:
  - Status
  - Company
  - Role
